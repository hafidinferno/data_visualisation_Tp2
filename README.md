Visualize Data with a Bar Chart ‚Äî FCC D3 Project

This project is part of the FreeCodeCamp Data Visualization Certification.
The goal is to build a responsive and interactive D3.js bar chart displaying the GDP of the United States, using real data from the BEA (Bureau of Economic Analysis).

This version includes improvements for readability, aesthetics, responsiveness, and modern UI design.

Live Demo

  FreeCodeCamp Project Requirements: https://www.freecodecamp.org/learn/data-visualization/

Features
Fully compliant with all 13 FreeCodeCamp User Stories:

Title with id="title"

X-axis and Y-axis created using D3

Ticks on both axes

One <rect> bar per data point

Bars aligned with axes using D3 scales

data-date and data-gdp attributes on each bar

Tooltip with id="tooltip" appearing on hover

Tooltip includes a matching data-date

Additional Enhancements (Bonus)

Smooth fade-in animation for bars

Modern tooltip with transition & subtle movement

Vertical & horizontal gridlines

Glassmorphism container design

Highlight effect on bar hover

Responsive typography for small screens

Clickable BEA data source link

Technologies Used

HTML5

CSS3 (Glassmorphism, modern UI)

JavaScript ES6

D3.js v7 (scales, axes, transitions)
